Soul

roles: technical artist, programmer, graphics programmer

Soul is the masters thesis project for Game/Industrial Designer Kees Luyendijk (Indiana University, 2017). As an artist-programmer hybrid I was asked to assist in creating a unique abstract visual style that could convey the ideas, themes, and information the game was meant to portray, as well as assist in the creation of levels and environments.

As a programmer I was in charge of creating tools for procedural mesh generation to aid the level creation process, a dynamic 2D light system, and the behaviors of various small environmental components. I also utilized the lighting and post-processing tools available in the Unity game engine to create a particular atmosphere which was both visually interesting and effective in conveying important information to the player. In order to achieve many effects I wrote various pieces of rendering code using Unity's Cg shader language, creating reusable material assets for the objects in the game world.
