Meteor Dance Wave VR

role: solo developer

Created as a first dive into Unreal Engine 4, Meteor Dance Wave VR is an active virtual-reality rhythm game prototype featuring vibrant visuals and a retro synthwave aesthetic.

Features
  • Central beat keeping system
  • Central sound visualization system
  • Dyamic materials (responding to beat & frequency amplitudes)
  • Runtime processing of course file data
Tools Used
  • Unreal Engine 4 (Blueprint, Materials, C++)
  • Photoshop (Textures)
  • Blender (Modeling)


Project Duration : 1 month (July - August 2017)
