PixelSprite is a Unity package which aims to bridge the gap between modern rendering techniques and low-resolution stylized graphics. The main function of the plugin is to take 3D meshes, vector art, particle systems, or any other renderable object in Unity and transform it into pixel art dynamically in real time.
