Hyperspace Drifter is a classic vertically-scrolling shoot 'em, as well as my first project using the open-source Godot game engine. With health as your only resource you must balance offensive capabilities, time manipulation, and the ability to construct drones all while avoiding a constant barrage of enemies.

Responsibilities
Features
  • Custom procedural geometry generation for trails and UI
  • Extensible component framework for behavior composition
  • Flocking system for enemy movement supporting group cohesion, obstacle avoidance, and player tracking
Tools Used
  • Godot, GDScript, GLSL (gameplay, visuals)
  • Bfxr (sound effects)
  • Blender (3D assets)
  • Photoshop (textures)
