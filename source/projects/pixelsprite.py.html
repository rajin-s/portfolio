Features
  • Realtime dynamic pixelization of any renderable object in Unity
  • Stylization options including outlines and palette swapping
  • Fine-grained control over how generated graphics are drawn to the screen
  • Additional 2D tools including bendable objects and vertex snapping
Tools Used
  • Unity, C#, CG (pipeline, shaders)
  • Photoshop (icons, branding)

PixelSprite is a Unity package which aims to bridge the gap between modern rendering techniques and low-resolution stylized graphics. The main function of the plugin is to take 3D meshes, vector art, particle systems, or any other renderable object in Unity and transform it into pixel art dynamically in real time.
